### API 变更

#### `POST /api/materials`
- 无变更（保持原有创建功能）

#### 新增 `PUT /api/materials`
- 请求体：`{ id, name?, description?, category? }`
- 更新指定材料的信息

#### 新增 `DELETE /api/materials?id=<id>`
- 删除指定材料及其关联的色卡

#### `POST /api/materials/[materialId]/swatches`
- 无变更（保持原有创建功能）

#### 新增 `PUT /api/materials/[materialId]/swatches`
- 请求体：`{ id, color_name?, color_code?, description?, price?, stock? }`
- 更新指定色卡的信息

#### 新增 `DELETE /api/materials/[materialId]/swatches?id=<id>`
- 删除指定色卡及其关联的产品图片

### 使用说明

1. **编辑材料**：
   - 鼠标悬停在材料列表项上
   - 点击右上角的 ✏️ 按钮
   - 修改信息后点击"保存"

2. **删除材料**：
   - 鼠标悬停在材料列表项上
   - 点击右上角的 🗑️ 按钮
   - 确认删除（⚠️ 会同时删除该材料下所有色卡和产品图片）

3. **编辑色卡**：
   - 选择材料后，鼠标悬停在色卡卡片上
   - 点击右上角的 ✏️ 按钮
   - 修改信息后点击"保存"

4. **删除色卡**：
   - 选择材料后，鼠标悬停在色卡卡片上
   - 点击右上角的 🗑️ 按钮
   - 确认删除（⚠️ 会同时删除该色卡的所有产品图片）

### 技术细节

- 级联删除：删除材料时自动删除关联的色卡，删除色卡时自动删除关联的产品图片
- 乐观更新：编辑后即时刷新列表
- 悬停交互：操作按钮默认隐藏，悬停时显示，保持界面简洁
